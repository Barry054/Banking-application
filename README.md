# Banking-application

A backend banking system built with **Java Spring Boot** that showcases secure APIs, database management, and real-world backend functionality.

---

## Overview
This project is a fully functional backend for a banking application. It allows users to register, authenticate, manage accounts, perform transactions, and generate bank statements in PDF format.  

It demonstrates clean backend architecture, secure JWT-based authentication, and integration with MySQL for persistent data storage. Swagger documentation is included for all API endpoints.

---

## Key Features
- User registration and login with **JWT authentication**
- Account management: create, read, update, delete accounts
- Transaction management: deposits, withdrawals, transfers
- Email notifications for account activities
- PDF generation of bank statements
- REST API documented using Swagger
- Input validation and error handling for reliable operations

---

## Technologies
- **Java 21**  
- **Spring Boot 3.2.5**  
- **Spring Security + JWT** for authentication and authorization  
- **Spring Data JPA** for ORM  
- **MySQL** as the database  
- **Spring Boot Mail** for notifications  
- **Swagger (springdoc-openapi)** for API documentation  
- **iTextPDF** for PDF generation  
- **Lombok** to reduce boilerplate  
- **JUnit / Spring Boot Test** for testing
